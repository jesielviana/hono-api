API Sandbox do Prof. Jesiel Viana

Objetivo da API

A API Sandbox foi desenvolvida para fornecer um ambiente de prática para os alunos das disciplinas de desenvolvimento web. Ela permite a integração com aplicações front-end, facilitando o aprendizado sobre consumo de APIs REST.

Uso da API

A API está disponível para os alunos realizarem requisições HTTP a partir de suas aplicações front-end, permitindo a prática de integração e manipulação de dados dinâmicos. É importante que os alunos utilizem a API de forma responsável, garantindo que as operações realizadas sejam adequadas ao contexto de aprendizado.

Atualmente, a API oferece endpoints para operações CRUD (Create, Read, Update, Delete) relacionadas a filmes.

Por favor, leia a documentação da API para obter detalhes completos sobre os endpoints, parâmetros e exemplos de uso.
